# 开发场景

## 步骤一：登录
### 1. 进入登陆界面
打开 [腾讯云 CloudStudio - 云端开发，化繁为简](https://cloudstudio.net/ "腾讯云 CloudStudio - 云端开发，化繁为简")，页面右上角会显示 **「登录/注册」** 按钮，点击进入登录界面。  

> 若直接访问平台功能页（如模板中心、课程中心），未登录状态下会自动跳转至登录界面。 

   <!-- <div style={{ width: '200px' }}>![](images/20250901_204703.png)</div> -->



<!-- <div style={{ width: '600px' }}>![](images/20250901_204736.png)</div> -->
 
### 2. 登陆与退出 

#### 2.1 登陆  

- **操作步骤：** 点击登录界面的 **「微信」** 按钮 → 弹出微信二维码 → 用微信扫描二维码（电脑端微信已登录时支持快捷确认） → 在微信中点击 **「确认登录」** → 页面自动刷新并完成登录。

  <!-- <div style={{ width: '800px' }}>![](images/微信登录.gif)</div> -->

> 这里仅对推荐方式做说明，如有其他登陆需求可查看 [登陆方式](../../product_use/account/login/login.md) 加以说明

### 2.2 **退出**：  

- **操作步骤：** 点击页面右上角的 **「用户头像」** → 在下拉菜单中选择 **「退出登录」** → 确认后即可退出当前账号，返回登录界面。

### 应用创建
应用创建包含 **「从模版创建」、「从 Git 仓库导入」、「二次开发」**

#### 如何选择？
- **「从模版创建」**：基于平台预设的「成品化基础框架」快速搭建应用，模版已包含通用功能（如基础页面布局、登录模块、数据库连接逻辑等），无需从零写代码。
- **「从 Git 仓库导入」**：将你已在 Git 仓库（如 CNB、GitHub、GitLab、Gitee）中编写好的完整代码项目，直接导入平台进行部署，本质是 “复用已有代码资产”，平台仅负责环境配置和运行。
- **「二次开发」**：将已有在 CloudStudio 社区中创建好的公开他人应用以及个人应用进行二次开发。

详细应用创建请查看 [应用创建](../../product_use/application/create-application/create-application.md)
